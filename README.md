# Gorilla Quest

A simple browser game where a Gorilla must collect bananas and defeat a Snake boss to become the Jungle King.

## Gameplay
- Move left and right to navigate the level and collect bananas.
- Jump to avoid obstacles and reach higher platforms.
- In the boss fight, throw bananas at the Snake to defeat it.

## Controls
- Left Arrow: Move left  
- Right Arrow: Move right  
- Space: Jump  
- D: Throw bananas (boss fight only)

## Running locally
1. Clone or download the repository.
2. Open `index.html` in a modern web browser (Chrome, Firefox, Edge, etc.).
   - The entire game is contained in a single HTML file (`index.html`) and uses JavaScript for the game logic.
